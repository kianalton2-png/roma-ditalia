## Idea 1: "Old World Trattoria" — Warm Parchment & Deep Burgundy

**Design Movement**: Italian Rustic Modernism — the warmth of a 1960s trattoria with refined typographic clarity.

**Core Principles**:
- Warmth over coldness: every surface feels aged, lived-in, and welcoming
- Typography as decoration: oversized Italian script headings act as visual anchors
- Asymmetric storytelling: sections break the grid with diagonal cuts and offset imagery
- Color as emotion: deep burgundy and warm cream evoke Chianti and candlelight

**Color Philosophy**:
- Background: warm parchment/cream `#F5EDD8`
- Primary: deep Chianti burgundy `#7B1D2A`
- Accent: aged gold `#C9973A`
- Text: dark espresso `#2C1810`
- Emotional intent: warmth, heritage, appetite stimulation

**Layout Paradigm**:
- Hero: full-bleed image with diagonal text overlay, restaurant name in large italic serif
- Menu sections: alternating left/right image + text blocks with slight rotation
- Asymmetric card grid for menu items with torn-paper edge dividers

**Signature Elements**:
- Torn/rough-edge paper texture dividers between sections
- Italic serif drop caps on section intros
- Thin gold rule lines as decorative separators

**Interaction Philosophy**:
- Hover reveals on menu items: price fades in, description slides up
- Smooth parallax on hero image
- Section entrances with gentle fade-up

**Animation**:
- Hero text: staggered letter-by-letter reveal on load
- Menu cards: fade-up with 80ms stagger
- Nav links: underline draw from left on hover

**Typography System**:
- Display: Playfair Display (italic, bold) — for headings and restaurant name
- Body: Lora (regular, 400) — for descriptions and menu text
- Accent: Cormorant Garamond (light italic) — for taglines and quotes

## Idea 2: "La Dolce Vita" — Cinematic Italian Poster Style

**Design Movement**: Mid-century Italian graphic design meets contemporary editorial layout

**Core Principles**:
- Bold contrast: deep forest green against warm ivory creates dramatic visual tension
- Poster-style hierarchy: large typographic blocks command attention like vintage Italian posters
- Generous negative space: breathing room between elements creates elegance
- Authentic photography: real restaurant photos as the primary visual language

**Color Philosophy**:
- Background: warm off-white `#FAF6EE`
- Primary: deep forest green `#1A3A2A`
- Accent: terracotta red `#C0392B`
- Gold detail: `#D4A843`
- Emotional intent: sophistication, authenticity, appetite

**Layout Paradigm**:
- Hero: split layout — left half deep green with large white title, right half full restaurant photo
- Horizontal scrolling menu categories with large category labels
- Staggered two-column menu layout with generous spacing

**Signature Elements**:
- Large Roman numerals as section markers (I, II, III)
- Thin horizontal rules with centered diamond ornaments
- Circular photo frames for gallery images

**Interaction Philosophy**:
- Menu category tabs with smooth sliding indicator
- Photo gallery with lightbox on click
- Sticky navigation that transitions from transparent to solid green

**Animation**:
- Hero: slide-in from opposite directions (text from left, image from right)
- Menu items: subtle scale on hover (1.02x)
- Section transitions: horizontal wipe reveals

**Typography System**:
- Display: Cormorant Garamond (bold, 700) — for restaurant name and section headers
- Body: Source Serif 4 — for menu descriptions
- UI: DM Sans — for navigation and labels

## Selected Design: **Idea 2 — "La Dolce Vita"** (Cinematic Italian Poster Style)

**Rationale**: The deep forest green + warm ivory + terracotta red palette creates a sophisticated, appetite-stimulating palette that honors the restaurant's 60+ year heritage without feeling dated. The editorial poster-style layout sets Roma D'Italia apart from generic restaurant websites while remaining highly readable and functional. The split hero layout immediately communicates the restaurant's character through photography.
